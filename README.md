# Dynamic-sales-Dashboard
 Built a dynamic sales dashboard in Excel to analyze sales and profit across products, regions, and time. Used Pivot Tables, charts, and slicers for interactivity. Enabled filtering by category, country, and quarter, highlighting trends, top products, and key insights for better decision-making.
 
# Key insights
- The *Running Shoes* category achieved the highest sales (1792 units).  
- Profit peaks* during mid-year months (May–August).  
- *Low-priority orders* make up the majority (about 60% of total).  
- *Fashion* dominates across top product segments.
  
# Tech Stack
Tools
*Microsoft Excel* | Data visualization and dashboard creation |
*Pivot Tables & Charts* | Dynamic data summarization |
*Slicers* | Interactive filtering |
*Conditional Formatting* | Highlight performance metrics |
*Data Cleaning (Excel functions)* | Ensures accurate analysis |
